RAG for News Classification
and Alerts System (GenAI Project)

Automating real-time news monitoring, classification, and targeted alerts using Retrieval-Augmented Generation

Project Overview

This project is a scalable RAG system designed to ingest, process, and classify high-volume news streams in real-time. By leveraging generative AI, it provides automated, context-aware summaries and alerts, significantly reducing manual monitoring efforts.

Problem Statement

Manual tracking of relevant news across numerous sources is time-consuming and error-prone. The challenge was to build an intelligent system that could accurately filter noise and deliver critical information to stakeholders instantly.

Solution Implemented

The solution uses LangChain for orchestration. News articles are chunked, embedded using a Transformer model, and stored in Pinecone. A retrieval chain fetches relevant context for each query, and an LLM (GPT-4) generates a concise summary and classification. The system is exposed via a FastAPI endpoint.
