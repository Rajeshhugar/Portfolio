Analyzing customer feedback from Play Store reviews using advanced machine learning techniques.

Rajesh Hugar, Machine Learning Engineer

  • Data Collection

Overview

The objective of understand nuanced customer sentiment towards specific app features, and to leanr more about sentiment towards specific app features. This helps in understanding customer needs and equirements.

Data Collection & Preprocessing

  • Source: Play Store Reviews.
  • Data scrapped using Python (Selenium/BeautifulSoup).
  • Preprocessing steps included text cleaning, tokenization, stop word removal, and stemming/lemmatization.

Methodology

  • Feature Engineering: Utilized pre-trained Word Embeddings (Word2Vec, GloVe) to capture semantic relationships.
  • Model Selection: Implemented XGBoost for its gradient boosting capabilities and performance on structured text data.
  • Model Training & Evaluation: Trained on a labeled dataset, optimized hyperparameters, and evaluated using accuracy, precision, recall, and F1-score.
