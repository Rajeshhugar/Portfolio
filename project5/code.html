Project Detail: Cross Sell Prediction

Predicting High-Potential Leads for Targeted Sales Campaigns

The Challenge

The objective was to identify existing customers with a high probability of purchasing an additional product, specifically a premium credit card. The primary goal was to optimize marketing efforts and increase cross-sell conversion rates.

The Solution

Developed an end-to-end predictive model using ensemble machine learning techniques (Random Forest, Gradient Boosting). The pipeline involved extensive feature engineering from customer demographics, transaction history, and engagement data, and addressing class imbalance with SMOTE.
